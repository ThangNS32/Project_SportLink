name: Show code in chat, don't write to files directly
description: User wants code shown in chat so they can add it themselves — do not write directly to files
type: feedback
---

Khi viết code mới hoặc sửa code, hãy paste code vào chat thay vì dùng Write/Edit tool để ghi thẳng vào file.

**Why:** User muốn tự tay thêm code vào để hiểu từng dòng, không muốn file bị sửa mà không qua tay mình.

**How to apply:** Mọi lần viết code mới — show code block trong chat. Chỉ dùng Write/Edit khi user nói rõ "hãy sửa file" hoặc "hãy thêm vào file".
